# Agro Forte, Futuro Sustentável 🌾🚜

Este é um site interativo desenvolvido para conscientizar e demonstrar como o agronegócio moderno e tecnológico consegue produzir em alta escala mantendo o respeito e a preservação do meio ambiente.

## 🚀 Funcionalidades
- **Interface Responsiva:** Design adaptável para celulares, tablets e computadores.
- **Painel de Pilares:** Cards interativos que detalham as boas práticas do setor ao clique do usuário.
- **Simulador Agro-Sustentável:** Uma ferramenta simples em JavaScript para comparar o impacto ambiental e produtivo entre técnicas tradicionais e modernas.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 Como rodar o projeto localmente
1. Faça o clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente em qualquer navegador web.
